Objective

Design a Bluetooth-controlled system to switch a device ON and OFF using Arduino.

Components Used

Arduino UNO

LED

220Ω Resistor

Jumper Wires

Working Principle

The Arduino reads serial input (simulating Bluetooth communication).
When '1' is received, the LED turns ON.
When '0' is received, the LED turns OFF.

Simulation Platform

Tinkercad Circuits
