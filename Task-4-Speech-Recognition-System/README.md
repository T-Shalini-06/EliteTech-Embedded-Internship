Objective

Build a basic speech recognition system for command-based control of devices using an embedded board.

Components Used

Arduino UNO

LED

220Ω Resistor

Jumper Wires

System Design

In a real system, voice input is converted to text using a speech recognition module.
In this simulation, voice commands are represented as text commands entered via Serial Monitor.

Working Principle

When the command “on” is received, the LED turns ON.
When the command “off” is received, the LED turns OFF.

Simulation Platform

Tinkercad Circuits
